모듈?  변수 그리고 함수등을 이용해서 
자주사용 하는 코드를 이미 만들어 놓은 파일   (0000000.py)

모듈을 모아 놓은것?? -> 패키지 ( 모듈의 상위개념으로 쉽게 이해하자면 모듈을 모아놓은 폴더)

> 모듈이 내 컴퓨터 폴더 안에 없다면
    -> pip install 설치 후 모듈 불러오기

> 모듈사용때 import 해서 사용하면 된다.

예시 
> import 모듈이름1 모듈이름2...
> import 모듈이름  as 별칭(alias)
> from 패키지이름 import 모듈이름 (해당 패키지폴더 안에 있는 모듈)
> from 모듈이름 import 함수 or 변수이름 
> from 모듈이름 import * 모듈안에 있는 모든 함수/변수들
> from 패키지이름.모듈이름 import 함수 or 변수이름


주의점
> import를 할 때는 항상 모듈이나 패키지만 가능하다
   즉, 클래스나 함수만을 import 할 수 없다.
